# Recommended Extra Files (For Report Quality)

## 1) attack-claim-boundary.md
Purpose: one-page statement clarifying what is claimed, what is not claimed, and why.

## 2) defense-qa-quick-answers.md
Purpose: short evidence-backed responses for likely tutor questions.

## 3) evidence-citation-map.md
Purpose: map every important report sentence to a code location.

These files reduce Q&A risk and improve consistency between report, presentation, and oral defense.
